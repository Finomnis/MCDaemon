MCDaemon
========

A Java Daemon for automatic Minecraft server maintenance

_WORK IN PROGRESS_

Features:
---------

-	Automatic initialization and update
-	Automatic crash detection and restart
-	Scheduled call of an external backup script


Supported Minecraft-Versions:
-----------------------------

-	Vanilla
-	Bukkit
-	FTB


Supported Modes:
----------------

- Directly run jar as process (Platformindependant)
- Run as Daemon via jsvc (Unix)
- Run as Service via procrun (Windows)
